Overview
This project explores customer data to identify key business insights related to:
- Customer distribution
- Revenue by category
- Sales by age group
- Subscription patterns

##  Tools Used
- Power BI  
- SQL  
- Python (Pandas)  
- Google Colab  

##  Key Highlights
- Total Customers: 3,863  
- Average Purchase Value: $59.70  
- Average Review Rating: 3.75  
- Clothing category generates the highest revenue  
- Senior age group contributes the most sales and revenue  

##  Objective
To transform raw customer data into meaningful insights that support data-driven decision-making.
